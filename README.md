# Simple-Sine-Map
Simple Coding of the examples in Unit 2 of "Introduction to Complexity" https://www.complexityexplorer.org/courses/89-introduction-to-complexity

I have made a simple Python script with the functionaility as the NetLogo model provided on the course, "SineMap". Some models with fix point attractors, and some period attractors. The finaly a model with "Sensitive Dependence on Initial Conditions", Chaos.
